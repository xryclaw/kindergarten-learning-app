## 一、平台定位

为幼儿园大班孩子设计的家庭学习辅助工具，支持每日知识点课后复习。内容涵盖：**识字、拼音、数学、故事、Scratch编程启蒙**。

核心特色：
- 游戏化学习
- 进度追踪
- 错题归集与复练
- 支持家长/老师后台管理学习内容

## 三、前端学习模块（10个 Tab）

### 1. 汉字学习 `CharacterLearning.vue`
功能：
- **今日故事**：展示当天要学习的古诗/故事（如《江南》）全文阅读
- **认字练习**：当日生字以卡片形式展示，点击可朗读（预留接口）
- **书写练习**：田字格书写提示动画，展示笔画顺序
- **测验游戏**：
  - 从生字库中随机出题
  - 听音/看字选义
  - 答对得分，答错自动归入错题本
- **进度追踪**：显示当前模块完成百分比

---

### 2. 拼音练习 `PinyinPractice.vue`
功能：
- 展示当天拼音知识点（如复韵母 üe / ie / 整体认读 ye / yue）
- **四声标调练习**：拖拽或点击选择正确声调
- **拼音配对游戏**：将拼音和汉字图片连线配对
- **拼读挑战**：随机抽取拼音进行拼读测验
- 记录学生答题正确率与用时

---

### 3. 数学练习 `MathPractice.vue`
功能：
- **20以内应用题**：
  -  keyword 高亮（"一共"提示加法，"剩下"提示减法）
  - 特殊标注"求相差"类题目，孩子易错点
- **图形组成**：拖拽几何图形拼成指定图案
- **等分练习**：切分披萨/蛋糕的图形化互动
- 每道题提交后会判断对错，错题自动记录

---

### 4. 故事阅读 `StoryModule.vue`
功能：
- 每日故事全文展示（如《江南》）
- **生字高亮**：故事中当日生字用不同颜色标注
- **阅读理解题**：每读完一个故事后弹出 3-5 道选择题
- 题型包括：情节理解、人物关系、字词含义

---

### 5. Scratch编程 `ScratchModule.vue`
功能：
- 针对幼儿园孩子的**图形化编程启蒙**
- 展示基础积木块（移动、转向、重复、声音）
- **小动画演示**：看代码运行结果（小猫移动、画正方形等）
- **顺序排列游戏**：把打乱的积木按正确顺序排列
- 不使用真实 Scratch 运行环境，以可视化教学为主

---

### 6. 互动测验 `InteractiveQuiz.vue`
功能：
- 跨学科的**综合挑战**
- 混合题型：识字、拼音、数学
- 限时答题，答对加分
- 测试结束后展示总分与排名

---

### 7. 学习卡片 `LearningCards.vue`
功能：
- 所有已学知识点的**复习卡片**
- 分类浏览：汉字卡片、拼音卡片、数学公式卡片、故事卡片
- **翻转记忆**：正面问题，背面答案，类似 Anki
- 支持按时间筛选（本周/本月/全部）

---

### 8. 学习成就 `GamificationView.vue`
功能：
- **积分系统**：每次完成练习获得积分
- **等级体系**：积分累积提升等级
- **连续打卡**：记录每日学习天数 streak
- **勋章墙**：12枚默认勋章
  - 初次登录、识字达人、拼音高手、数学之星、故事大王等
  - 完美答题 streak、连续打卡 7 天等
- **排行榜**：按积分/等级/连续天数显示榜单

---

### 9. 错题复习 `ReviewSystem.vue`
功能：
- 收集所有模块的错题到统一题库
- **分类筛选**：按学科（识字/拼音/数学/故事/Scratch）查看
- **智能复习推荐**：基于艾宾浩斯遗忘曲线的 1/3/7/14/30 天推荐
- **重练模式**：错题重做，连续答对 3 次标记为"已掌握"
- **统计看板**：本周错题数、已掌握数、待复习数

---

### 10. 内容管理 `AdminPanel.vue`
功能：
> 仅限有管理权限的账号使用

- **知识点 CRUD**：
  - 新增/编辑/删除每日学习内容
  - 分类：character / pinyin / math / story / scratch
- **批量导入**：支持 JSON 格式批量导入题目和知识点
- **难度等级**：为每道题设置难度（1-3 星）
- **上下线开关**：控制内容是否对学生可见
- **内容排序**：拖拽调整展示顺序

## 四、后端能力

### 技术栈
| 层级 | 技术 |
|------|------|
| 后端框架 | Fastify (Node.js) |
| 数据库 | SQLite |
| 认证 | JWT + httpOnly Cookie |
| 限流 | `@fastify/rate-limit` |

### API 模块

#### 4.1 认证模块 `/api/v1/auth`
- `POST /login` - 家长登录（已关闭注册）
- `POST /logout` - 登出
- `GET /me` - 获取当前用户信息

#### 4.2 学生管理 `/api/v1/students`
- `GET /` - 获取当前家长下的所有孩子
- `POST /` - 创建孩子档案
- `PUT /:id` - 更新孩子信息
- `DELETE /:id` - 删除孩子档案

#### 4.3 学习记录 `/api/v1/learning`
- `POST /records` - 提交学习进度
- `GET /records` - 查询学习记录
- `POST /mistakes` - 记录错题
- `GET /mistakes` - 获取错题列表
- `DELETE /mistakes/:id` - 删除已掌握错题
- `GET /progress/:studentId` - 查询某位学生的整体进度

#### 4.4 内容管理 `/api/v1/content`
- `GET /topics` - 获取所有知识点
- `POST /topics` - 新增知识点（需管理员权限）
- `PUT /topics/:id` - 更新知识点
- `DELETE /topics/:id` - 删除知识点
- `POST /import` - JSON 批量导入

#### 4.5 游戏化 `/api/v1/gamification`
- `GET /stats/:studentId` - 查询学生积分/等级/streak
- `GET /badges/:studentId` - 查询已获得勋章
- `POST /events` - 触发事件（完成练习、获得勋章等）
- `GET /leaderboard` - 排行榜数据
